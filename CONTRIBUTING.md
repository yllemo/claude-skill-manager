# Contributing to Claude Skill Manager

Thank you for considering contributing to Claude Skill Manager! 

## How to Contribute

### Reporting Issues
- Use the GitHub issue tracker to report bugs or request features
- Include detailed information about your PHP version, web server, and steps to reproduce
- Check existing issues before creating a new one

### Development Setup

1. Fork the repository
2. Clone your fork:
   ```bash
   git clone https://github.com/YOUR-USERNAME/claude-skill-manager.git
   ```
3. Follow the installation instructions in the README
4. Create a feature branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```

### Code Style
- Follow PSR-12 coding standards for PHP
- Use meaningful variable and function names  
- Include comments for complex logic
- Keep functions focused and single-purpose

### Testing
- Test your changes with different PHP versions (8.1+)
- Verify functionality works on different web servers
- Test file upload/download functionality
- Check that authentication works properly

### Pull Request Process

1. Make sure your code follows the existing style
2. Update documentation if needed
3. Add your changes to CHANGELOG.md
4. Commit with clear, descriptive messages
5. Push to your fork and create a pull request

### Pull Request Guidelines

- **Title**: Use clear, descriptive titles
- **Description**: Explain what changes you made and why
- **Testing**: Describe how you tested the changes
- **Breaking Changes**: Clearly mark any breaking changes

## Questions?

Feel free to open an issue for any questions about contributing!
